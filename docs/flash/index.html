⚡ Flasher le M5Stack

Installe le firmware vibe-m5stack sur ton M5Stack Fire en un clic, sans PlatformIO ni toolchain. Il suffit d'un navigateur moderne (Chrome, Edge) et du driver USB CP210x.

✅ Prérequis

  • ✓ Navigateur Chromium (Chrome, Edge, Brave)
  • ✓ Port série disponible
  • ? Driver CP210x installé
  • ✓ M5Stack Fire connecté par USB
Sélectionne ton port COM et clique sur le bouton pour commencer.

📝 Instructions

  1. Branche ton M5Stack Fire via USB (câble data, pas charge-only)
  2. Installe le driver CP210x si nécessaire :
    • Windows : Télécharger ici
    • Linux/Mac : Déjà intégré dans le noyau
  3. Clique sur le bouton « Flasher le M5Stack » ci-dessus
  4. Sélectionne ton port COM (ex: COM8 sur Windows, /dev/ttyUSB0 sur Linux/Mac)
  5. Attends la fin du flash (~30 secondes)
  6. Ton M5Stack redémarre automatiquement et affiche le chat Mistral
